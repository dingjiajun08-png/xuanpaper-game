### 禁止使用
- 纯红 `#FF0000` 及大面积高饱和红色
- 霓虹荧光色（饱和度 > 90% 的纯色）
- 渐变、发光、柔化效果
- 现代商业街风格（LOGO 牌匾、LED 灯箱）
- 赛博朋克元素

## 二、像素网格规范

### 分辨率标准

| 素材类型 | 像素尺寸 | CSS 显示大小 | 备注 |
|---------|---------|------------|------|
| 小物件/装饰 | 16×16 | 32~48px | 落叶、纸屑、图标 |
| 小摊位道具 | 32×32 | 48~64px | 砚台、纸卷、陶罐 |
| 摊位整体 | 64×48 | 96~128px | 宣纸摊、文房摊 |
| 晾纸架（竖向） | 48×80 | 96×160px | 竖向摊位 |
| NPC 角色 | 24×32 | 48×64px（×2缩放） | 与主角精灵同规格 |
| 前景遮挡 | 任意宽×64h | 全宽 | 屋檐、竹叶、顶棚 |
| UI 弹窗边框角块 | 8×8 | 9-slice 拉伸 | |
| UI 图标 | 16×16 | 32px | 金币、声望、毛笔 |

### 导出规则
- 格式：**PNG-24 + Alpha 透明**
- 不导入背景色，保持纯透明背景
- CSS 已配置 `image-rendering: pixelated`，不使用双线性插值
- 命名格式：`[类型]_[名称]_[状态/帧].png`

### 命名示例
```
stall_xuanzhi_idle.png       宣纸摊（静止）
stall_wenfang_idle.png       文房四宝摊（静止）
npc_vendor_down_idle.png     摊主朝下待机
npc_vendor_down_walk.png     摊主朝下行走
npc_visitor_01_idle.png      游客 01 待机
deco_lantern_idle.png        灯笼静止
deco_lantern_sway.png        灯笼摆动帧
deco_flag_fly_01.png         旗幡飘动帧1
floor_stone_tile.png         石板路瓦片
floor_scatter_leaf_01.png    落叶散落01
fg_eave_left.png             屋檐前景左段
fg_bamboo_01.png             竹叶前景01
```

## 三、A. 集市摊位（stall_*）

### stall_xuanzhi — 宣纸摊
- 规格：64×48px（桌面+木架+顶棚一体）
- 顶棚颜色：`#edd8ad` 布料条纹 + `#2d241b` 4px 墨线边框
- 木架柱：`#72462d`
- 桌面：`#8a6848`
- 桌上要素：晾纸竹竿、卷宣（横放）、纸包（麻绳捆扎）、印章盒
- 晾纸：竹竿上悬挂 3-4 张白宣，纸色 `#f0e8c8`

### stall_wenfang — 文房四宝摊
- 规格：64×48px
- 背景布：`#c8b48a` 稻草黄条纹
- 要素：毛笔悬挂于笔架、砚台（`#4e3a28`）、墨锭（`#2d241b`）叠放、陶罐（`#9d4f34`）
- 桌腿用木色像素块

### stall_dryrack — 晾纸架（竖向）
- 规格：48×80px
- 竹竿架：`#8a6848` 2px 横竿 + `#4a7c59` 竖竿
- 悬挂 3-4 张白宣，用绳结（`#c8b48a`）固定
- 纸张可加墨迹随机点缀（可选，每张不同）

### prop_shelf — 竹木架
- 规格：48×48px
- 横格分层 3 层，可叠放纸卷
- 木色：`#72462d`·`#8a6848` 两色交替像素

## 十、深度排序规则（z-index 层级）

```
地面/石板路         z=0
摊位底部/道具       z=1 ~ 玩家y值
NPC 游客           z=玩家y值
玩家角色            z=玩家y值+1
摊位顶棚/悬挂装饰   z=900
前景遮挡（竹叶/屋檐）z=1000
UI 层               z=2000+
```

> 深度排序已在 `game.js` 中实现（`updateDepthSortedEntities()`），新素材直接加入深度排序逻辑即可。

## 十一、动画帧规范

| 素材 | 帧数 | 帧间隔 | 备注 |
|------|------|--------|------|
| NPC idle（各方向）| 1帧 | — | 静止 |
| NPC walk（各方向）| 2帧 | 180ms | 与主角一致 |
| 灯笼摆动 | 2帧 | 800ms | 慢速 |
| 旗幡飘动 | 3帧 | 200ms | 风吹效果 |
| 落叶飘落（可选）| 4帧 | 150ms | CSS animation |

---

*本规范基于游戏现有 CSS 变量体系，所有新素材须严格遵守色值和像素规格，确保与现有地图美术风格统一。*
