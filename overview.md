# 《纸境千年》v2.0 改动报告

## 📋 四大改动概览

### 1️⃣ 订单系统扩展 — 12位有故事的NPC
- **从4个NPC扩展到12个**，每位NPC都有：
  - 专属名字和性格（王大爷、画师李姐、书院学生、赶考书生、日本茶人、瓷器商人、流浪诗人...）
  - 独特的对话引语（在订单卡片中显示）
  - 交付时的专属评价
  - 关联的宣纸科普知识
- 声望解锁系统：普通订单(0)→精品订单(25)→限时订单(45)→定制订单(55+)

### 2️⃣ 工坊小游戏深化 — 三阶段制纸流程
- **阶段1 - 浸泡控制**：进度条自动走动，在绿色区域(70-85%)点击停止，影响韧性和均匀度
- **阶段2 - 浓度平衡**：两根bar独立移动，需同时控制在绿色区域并保持1.5秒，影响吸墨性和白度
- **阶段3 - 抄纸**：保留原有指针停止判定
- 三个阶段综合计算最终纸张属性，报告显示每阶段评分

### 3️⃣ 科普内容注入 — 12张科普卡
- cardData从3张扩展到12张：
  - 原料类：青檀树皮、沙田稻草
  - 工艺类：抄纸手法、晒纸工艺、七十二道工序
  - 纸性类：吸墨性、生宣与熟宣、墨分五色
  - 文化类：纸寿千年、泾县宣纸、非遗传承、书画同源
- 工坊每次进入随机展示科普提示
- 订单交付后展示关联科普知识
- 完成订单随机解锁新科普卡

### 4️⃣ 升级UI美化
- grid卡片布局替代旧式列表
- hover上升效果 + 金色边框
- 已升级/可购买/锁定三种状态清晰区分
